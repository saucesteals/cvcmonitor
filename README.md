## Environment

#### **_`.env` supported_**

- `SEARCH_URL` - The url for your search on https://search.cvc.edu/search including filter queries
- `DISCORD_WEBHOOK_URL` - Discord webhook to send updates to

## Run

```sh
$ go run .
```
